利用 Hexo 和 Github 搭建静态博客

发表于 2017-06-25

为什么写博客？

   其实想搭博客很久了，本科时就尝试过用 github 提供的模板搭了静态的页面，但后面没有用过。一是界面实在太丑，二来虽有做学习笔记的习惯，但自觉拿不出手，只是自己在云笔记里记录着。
   转眼间研二已经快要过去了，又想起来创建博客的事。一来自己成长了不少，有一定的知识积累。二来这些年从别人的博客或者开源项目学到了很多东西，总觉得不能一味索取，应该与他人分享，同时参与开源项目，回馈社区。

为什么选择Github Pages + Hexo ?

  1. 免费，Github Pages 提供500M免费空间，并且不限流量
  2. Hexo 有本地服务器，可以在本机预览，管理也很方便
  3. Hexo 主题丰富，好看
